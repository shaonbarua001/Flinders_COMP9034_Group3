# AGENT Guide

## Language Requirement
This project is English-only.
Use English for all source code, comments, documentation, commit messages, and UI text.

## Directory Guide
- `design/`: Legacy design files. This directory is outdated and kept for archive/reference only. Do not use it as the current implementation source.
- `apps/api/db/`: Database SQL scripts (schema creation, initialization, and migration-related SQL).
- `apps/api/`: Backend service (APIs, business logic, and server-side implementation).
- `apps/web/`: Frontend application (web UI and interaction implementation).
